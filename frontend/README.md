# Focusnyx Frontend (PWA)

Next.js 14 frontend for the Focusnyx student life OS.

## Feature Areas
- Dashboard with daily progress and XP rewards
- Smart Academic Forge module
- Dopamine Detox Pomodoro module
- Smart Notes Vault module
- Student Finance Tracker module
- Wellness Shield module
- AI Behavioral Coach module
- 3-tier productivity analytics views

## Tech Stack
- Next.js 14
- NextAuth.js
- Supabase client SDK
- Recharts

## Start
```bash
npm install
npm run dev
```
